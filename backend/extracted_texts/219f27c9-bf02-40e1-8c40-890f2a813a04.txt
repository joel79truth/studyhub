1. Describe cell referencing types in excel with examples 
In Excel, cell referencing determines how a cell address behaves when copied or moved. There 
are three main types of cell referencing: 
1. Relative Cell Reference 
 Definition: Changes based on the position where it is copied or moved. 
 Example:  
o Formula in B2: =A2+10 
o If copied to B3, it changes to =A3+10. 
2. Absolute Cell Reference 
 Definition: Remains fixed when copied or moved by using $ symbols before the row and 
column. 
 Example:  
o Formula in B2: =$A$1+10 
o If copied to B3, it remains =$A$1+10. 
3. Mixed Cell Reference 
 Definition: Either the row or column is fixed while the other part changes. 
 Types:  
o Fixed column, changing row ($A1) 
o Fixed row, changing column (A$1) 
 Example:  
o Formula in B2: =$A2+10 
o If copied to C3, it changes to =$A3+10 (column remains A, row changes). 
These referencing methods are useful in formulas, especially when working with large datasets. 
2. HOW TO Sorting & filtering data IN EXCEL WITH EXAMPLES 
Sorting & Filtering Data in Excel 
Sorting and filtering are essential functions in Excel that help organize and analyze data 
effectively. 
 

1. Sorting Data in Excel 
Sorting allows you to arrange data in ascending or descending order based on one or more 
columns. 
Steps to Sort Data: 
1. Select the range of data you want to sort (e.g., A1:C10). 
2. Go to the "Data" tab. 
3. Click "Sort". 
4. Choose the column you want to sort by. 
5. Select either Ascending (A → Z) or Descending (Z → A). 
6. Click OK. 
Example: 
Name Age Salary 
John 25 5000 
Alice 30 7000 
Bob 22 4000 
 Sorting by Age (Ascending):  
o Bob → John → Alice 
 Sorting by Salary (Descending):  
o Alice → John → Bob 
 
2. Filtering Data in Excel 
Filtering allows you to display only specific data while hiding the rest. 
Steps to Apply a Filter: 
1. Select the dataset. 
2. Go to the "Data" tab. 
3. Click "Filter" (a small drop-down arrow appears on each column header). 
4. Click the arrow in the column header and select the criteria (e.g., filter Age > 25). 
5. Click OK to apply the filter. 
Example: 

Name Age Salary 
John 25 5000 
Alice 30 7000 
Bob 22 4000 
 Filter Age > 25:  
o Only Alice appears. 
 Filter Salary ≥ 5000:  
o John and Alice appear. 
 
Advanced Sorting & Filtering: 
 Sort by multiple columns: Sort first by "Salary" and then by "Age." 
 Custom Filters: Use options like “Greater than,” “Less than,” or “Contains.” 
 Clear Filters: Click Data → Clear Filter to remove filters. 
Sorting and filtering make it easier to manage large datasets and analyze information efficiently.

6. HOW TO fv, pv, pmt IN EXCEL WITH EXAMPLES 
Financial Functions in Excel: FV, PV, and PMT with Examples 
Excel provides built-in financial functions to calculate Future Value (FV), Present Value 
(PV), and Payment (PMT) for financial analysis. 
 
1. FV (Future Value) Function 
The FV function calculates the future value of an investment based on periodic, constant 
payments and a constant interest rate. 
Syntax: 
=FV(rate, nper, pmt, [pv], [type]) 
 rate → Interest rate per period 
 nper → Total number of payment periods 
 pmt → Payment per period (if applicable) 
 pv (optional) → Present value (initial investment) 
 type (optional) → 0 (end of period) or 1 (beginning of period) 
Example: 
You invest $1000 at an annual interest rate of 5% for 5 years with no additional payments. 
 Formula: =FV(5%/12, 5*12, 0, -1000, 0) 
 Result: $1,276.45 
 
2. PV (Present Value) Function 
The PV function calculates the present value of an investment based on periodic payments and a 
discount rate. 
Syntax: 

=PV(rate, nper, pmt, [fv], [type]) 
 rate → Interest rate per period 
 nper → Total number of payment periods 
 pmt → Payment per period 
 fv (optional) → Future value 
 type (optional) → 0 (end of period) or 1 (beginning of period) 
Example: 
You want to know how much you need to invest today to get $5000 in 5 years at an 8% annual 
interest rate. 
 Formula: =PV(8%/12, 5*12, 0, 5000, 0) 
 Result: -3,395.30 (You need to invest $3,395.30 today) 
 
3. PMT (Payment) Function 
The PMT function calculates the fixed payment required to repay a loan or investment. 
Syntax: 
=PMT(rate, nper, pv, [fv], [type]) 
 rate → Interest rate per period 
 nper → Total number of payments 
 pv → Loan or investment amount 
 fv (optional) → Future value (default is 0) 
 type (optional) → 0 (end of period) or 1 (beginning of period) 
Example: 
You take a loan of $10,000 at an annual interest rate of 6% for 5 years, and you need to 
calculate the monthly payment. 
 Formula: =PMT(6%/12, 5*12, -10000, 0, 0) 
 Result: $-193.33 (You must pay $193.33 per month)
